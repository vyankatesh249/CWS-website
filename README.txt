STEP 1 — TEST LOCALLY
----------------------
Double-click index.html → opens in your browser. Done.


STEP 2 — CONNECT GMAIL (Formspree) — 5 minutes
------------------------------------------------
1. Go to: https://formspree.io
2. Sign up FREE with your Gmail
3. Click "+ New Form" → name it "CWS Contact" → Create
4. Copy your form link e.g. https://formspree.io/f/abcdefgh
5. Open index.html in VS Code or any text editor
6. Find this line:
      action="https://formspree.io/f/YOUR_FORMSPREE_ID"
7. Replace YOUR_FORMSPREE_ID with your actual ID (e.g. abcdefgh)
8. Save the file.
Now every form submission goes to cwspersonalpodcast@gmail.com


STEP 3 — HOST ON NETLIFY (Free Forever) — 2 minutes
-----------------------------------------------------
1. Go to: https://netlify.com
2. Sign up FREE with your Gmail
3. Click "Add new site" → "Deploy manually"
4. Drag & drop this entire folder onto the page
5. You get a link like: https://random-name.netlify.app
6. To rename: Site Configuration → Change site name
   e.g. → https://cws-cusucceed.netlify.app

That link works FOREVER for free. No credit card needed.


NEED TO UPDATE THE WEBSITE LATER?
-----------------------------------
- Edit index.html or style.css
- Go back to Netlify → your site → Deploys
- Drag & drop the updated folder again
- Site updates in 30 seconds
